# Руководство пользователя To-Do List

## Как начать
1. Откройте приложение в браузере.
2. Зарегистрируйтесь: email и пароль (мин. 6 символов).
3. Войдите с теми же данными.

## Работа с задачами

### Создать
- Нажмите на синюю круглую кнопку `+`.
- Введите заголовок (обязательно), опционально описание и дату.
- Нажмите "Сохранить".

### Выполнить
- Нажмите на кружок слева от задачи.

### Редактировать
- Нажмите на текст задачи → измените → сохраните.

### Удалить
- Нажмите на иконку корзины 🗑️ → подтвердите.

### Поиск
- Введите слово в поле поиска (минимум 2 символа).

### Фильтры
- Используйте табы: "Все", "Активные", "Выполненные".

## Частые вопросы
**Что означают красные даты?** – Задача просрочена.

**Как выйти?** – Нажмите "Выйти" в правом верхнем углу.

**Почему меня выкинуло из системы?** – Сессия истекает через 60 минут.
