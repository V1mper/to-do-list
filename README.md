# To-Do Трекер (Python)

## Описание
Веб-приложение для управления задачами, написанное на FastAPI.

## Стек
- Backend: Python 3.11, FastAPI, SQLAlchemy 2.0, Alembic
- База данных: PostgreSQL
- Аутентификация: JWT (PyJWT + passlib[bcrypt])
- Frontend: React (или статика)
