### Техническая реализуемость (BE)
Все данные в прототипе можно отдать через API:

•	Список задач → GET /tasks

•	Чекбокс → PUT /tasks/{id} {is_done: true}

•	Форма → POST /tasks

•	Поиск → GET /tasks?search=...
